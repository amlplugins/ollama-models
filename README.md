# @amlplugins/ollama-models

Native `ollama` SDK re-exported under the `@amlplugins` namespace.

> Ollama Models — list/pull/push/copy/delete local model registry; show model info and Modelfile.

## Install

```bash
npm install @amlplugins/ollama-models
```

## Use

```ts
import { /* named exports from ollama */ } from "@amlplugins/ollama-models";
// or
import sdk from "@amlplugins/ollama-models";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
